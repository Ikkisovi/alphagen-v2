# Documentation Index

This directory collects all project documentation in one place.

- `CUSTOM_DATA_UPDATE_v0_10.md` – latest custom fundamental dataset procedure. Older versions are archived under `archive/custom_data_update/`.
- `DAILY_DATA_QUICKSTART.md`, `DAILY_DATA_SUPPORT.md`, `DAILY_DATA_UPDATE_SUMMARY.md` – references for maintaining the daily data ingestion pipeline.
- `DEBUGGING_FINDINGS_v0_11.md`, `IMPLEMENTATION_SUMMARY.md` – post-mortems and build notes for recent rollouts.
- `OBJECTSTORE_DYNAMIC_LOADING.md`, `ROLLING_UPGRADE_COMPLETE.md`, `VERIFICATION_AND_RERUN_GUIDE.md`, `QUICK_START.md` – operational guides for the rolling trainer and Lean deployment.
- `archive/` – temp notes, superseded documents (e.g., `v0.3.md`, older custom data updates). Move new superseded material here to keep the root list focused.

When publishing an updated doc, keep the newest revision in this folder and move previous iterations into the archive so the active guidance is easy to find.
